#  Smart Uber Demand Prediction System

##  Overview
This project predicts Uber ride demand based on time features using machine learning models.

##  Features
- Data preprocessing & feature engineering
- 20+ visualizations
- Multiple ML models (Linear, RF, XGBoost)
- Model comparison & selection
- Surge pricing logic
- Streamlit dashboard

##  Model Used
Best model selected based on MAE score.
